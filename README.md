
# PasswordTool
![Python](https://img.shields.io/badge/python-3.8%2B-blue) ![Status](https://img.shields.io/badge/status-active-success) ![Version](https://img.shields.io/badge/version-0.1-blue)

PasswordTool v0.1 is a simple Python CLI application for generating secure passwords and passphrases, checking password strength, and storing credentials locally. It provides multiple password generation modes with customizable options and basic clipboard support for quick copying.
## Installation
```bash
git clone https://github.com/realpnut/passwordtool
cd passwordtool
pip install -r requirements.txt
python main.py
```
